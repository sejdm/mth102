Expectation of a function of a random variable

5 March, 2025 (Wednesday)

 Last modified: 05:53 PM - 5 March, 2025

PDF version

There is still some confusion about why \(E[g(X)] = \sum g(x_i) p(x_i)\) and not \(E[g(X)] = \sum g(x_i) p(g(x_i))\). I think the indices in the proof may be an additional roadblock so let us try and see it with an explicit example that is typical of what happens in general.

Let us consider a random variable \(X\) that can take the values \(\{-2, -1, 1, 2\}\). Let us assume that the probability mass function \(p_X(-2) = 0.3\), \(p_X(-1) = 0.1\), \(p_X(1) = 0.2\), \(p_X(2) = 0.4\). So the expectation, \(E[X] = -2 \times 0.3 + -1 \times 0.1 + 1 \times 0.2 + 2 \times 0.4\). I am using the notation \(p_X(a)\) instead of just \(p(a)\) for the mass function to avoid confusion with another mass function which we will see shortly.

Let us now consider the function \(g(x) = x^2\). First, let us be clear about what we mean by \(E[g(X)]\). For that we need to be clear what we mean by the random variable \(g(X)\). Remember that \(X\) can take the values \(-2, -1, 1, 2\). So \(g(X)= X^2\) can take the possible values 1, or 4 because both \((-1)^2\) and \(1^2\) are 1, and both \((-2)^2\) and \(2^2\) are 4.

Let us denote this new random variable \(X^2\) by \(Y\). We are interested in \(E[Y]\). Since our random variable \(Y\) can take values \(1, 4\), \(E[Y] = 1 \times p_Y(1) + 4 \times p_Y(4)\). Notice that I am using \(p_Y\) instead of \(p\) so as not to confuse it with the mass function, \(p_X\), for the random variable \(X\).

But what does \(p_Y(1)\) mean for this random variable? By definition, \(p_Y(1) = P\{Y=1\}\), in other words the probability that \(Y\) takes the value 1. But that itself is the probability that \(X\) takes the value 1 or -1, i.e. \(p_Y(1) = P\{Y = 1\} = P\{X^2 = 1\} = P\{X = 1\} + P\{X=-1\}= 0.1 + 0.2\).

Similarly, by definition, \(p_Y(4) = P\{Y=4\}\), in other words the probability that \(Y\) takes the value 4. But that itself is the probability that \(X\) takes the value 2 or -2, i.e. \(P\{X = 2\} + P\{X=-2\}= 0.3 + 0.4\).

So the expectation \(E[Y] = 1 \times (0.1 + 0.2) + 4 \times (0.3 + 0.4)\). \[= 1 \times 0.1 + 1 \times 0.2 + 4 \times 0.3 + 4 \times 0.4\] \[= 1 \times p(-1) + 1 \times p(1) + 4 \times p(-2) + 4 \times p(2)\]

Now here is the crucial observation! \(1 = 1^2\), but 1 is also \((-1)^2\). Similarly, \(4 = 2^2\) but 4 is also \((-2)^2\). So, let us replace one 1 with \(1^2\) but the other with \((-1)^2\) (and do the same for 4) just to get a better looking expression! \[= 1^2 \times p_X(-1) + (-1)^2 \times p_X(1) + 2^2 \times p_X(-2) + (-2)^2 \times p_X(2)\].

As we can see in this example, \(E[g(X)]=\sum g(x_i) p(x_i)\)
